# 🌱 Smart Farmer – Arecanut Plant Disease Detection System

AI-powered web application for detecting Arecanut plant diseases using Deep Learning and providing disease descriptions, precautions, and solutions in both English and Kannada.

---

# 📌 Project Overview

This project uses a trained Deep Learning model to identify diseases in Arecanut plants from uploaded leaf images.

The system:
- Detects plant disease
- Displays confidence percentage
- Shows disease description
- Provides precautions
- Suggests fertilizer / treatment solutions
- Supports English and Kannada languages

---

# 🚀 Features

✅ Arecanut disease detection using AI  
✅ Deep Learning model (TensorFlow / Keras)  
✅ FastAPI backend  
✅ HTML/CSS/JavaScript frontend  
✅ English + Kannada language support  
✅ Farmer-friendly UI
